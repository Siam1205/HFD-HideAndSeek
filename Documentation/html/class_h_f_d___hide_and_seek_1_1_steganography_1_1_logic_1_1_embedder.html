HFD-HideAndSeek  1.1
HFD-HideAndSeek is an LSB-based Steganography tool developed for the purpose of teaching. It implements Sequential as well as Randomized Hide & Seek algorithms in order to hide arbitrary files inside BMPs or 24-bit PNGs. Therefore, it allows to use 1 up to 7 of the carrier image's bit planes (although this is not recommended at all). In addition, HFD-HAS facilitates to encrypt the messages before hiding with AES256-CBC and allows the user to display bit planes of an image so that he may apply a visual attack to potentially generated stego images himself. Lastly, the tool supplies the possibility to rate an image regarding its suitability for a specific message. Such a rating can predict whether the generated stego image is robust against statistical attacks which is the case when the rating reaches 90%.

Detailed Description

This class represents a message embedder which provides methods to embed or extract steganographical messages. Such messages can be any file. On the other hand, the medium a message can be hidden within must be a System.Drawing.Bitmap object which is available in the Format24bppRgb pixel format. Such images are most likely BMPs or PNGs.
